Config file for `i3` window manager
===================================

## Installation for Ubuntu 18.04 or newer

### Install `i3` window manager

```
$ sudo apt update
$ sudo apt install i3
```

### Dwonload config

```
$ git clone git@github.com:doncasper/i3.git ~/.config/i3
``` 

Aafter that you can relogin using `i3` windown manager.
